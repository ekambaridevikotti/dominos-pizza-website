# Domino's Pizza Website Layout

A responsive Domino's Pizza website layout built using HTML and CSS with Flexbox and Grid.

## Features

- Responsive navbar with navigation links
- Product cards section using CSS Grid  
- Menu items grid layout
- Download app section
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- Responsive Design

## Setup

1. Open `dominos.html` in your browser to view the website
